# fiber-comments
The repo adds useful information about creating a restful comment system in GO. This is built with ⚡️Fiber, MongoDB, and of course, with lots of :heart:.

## Features
- Restful API
- Dockerized
- Using ⚡️ Fiber framework
- Using MongoDB
- JWT Authentication
- Documented

## Usage
Follow this steps:
```bash
    cp .env.example .env
    make up
```
